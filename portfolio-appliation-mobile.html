Application Mobile

Création d’une application mobile de beatbox interactive, conçue pour les passionnés de musique et les amateurs de rythmes urbains. Cette application propose une interface intuitive où les utilisateurs peuvent créer leurs propres morceaux en superposant des sons de beatbox prédéfinis, avec la possibilité de les enregistrer et de les partager. Des fonctionnalités de mixage et d’effets sonores permettent d’ajuster les beats en temps réel, offrant une expérience immersive et créative. Un mode tutoriel guide les débutants pour apprendre les bases du beatboxing, tandis qu'un espace communautaire permet aux utilisateurs d’échanger leurs créations. Ce projet démontre mes compétences en développement mobile, ainsi qu’en conception d’interfaces interactives et ludiques.

Technologies utilisées: KIVY,et d'autres ( à compléter )
